# gclimb — Healthcare IoT Hub

## Goal
Build a well-structured SmartThings repository for healthcare IoT device handlers and SmartApps, replacing the old `.src`-nested structure with flat files, GitHub Actions CI, and Groovy linting.

## Why This Matters
Healthcare IoT devices need reliable, maintainable integrations. A clean codebase makes it easier to add new devices, catch bugs early, and onboard contributors.
